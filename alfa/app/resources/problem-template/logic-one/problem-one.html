Diketahui premis-premis berikut:

P1: Semua {{ p1-a }} pasti {{ p1-b }}
P2: {{ p2-a }} adalah {{ p1-a }}

Kesimpulan yang pasti valid dari kedua premis tersebut adalah

A. Beberapa {{ p2-a }} yang {{ p1-b }}
B. Ada {{ p1-a }}
C. Tidak ada jawaban yang tepat

===sepa-problem-soal===

Bentuk Semua A adalah B itu konsekuensinya apa pun yang termasuk A pasti dia adalah B. So jadi kalo {{ p1-a }} adalah {{ p1-b }} dan kita tau dari soal juga bahwa {{ p2-a }} adalah {{ p1-a }} ya jelas bahwa:
1. {{ p2-a }} itu ada
2. {{ p2-a }} adalah {{ p1-a }}
3. Maka {{ p2-a }} adalah {{ p1-b }}
